CRUD de Animales - Java / JavaFX

Introducción:

Este proyecto es mi primer CRUD desarrollado en Java, con interfaz gráfica en JavaFX y diseño de vistas utilizando Scene Builder.
Lo desarrollé con el objetivo de poner en práctica conceptos fundamentales de programación, especialmente de Programación Orientada a Objetos (POO), y también para mejorar mi forma de pensar, estructurar y resolver problemas mediante código.
La aplicación permite realizar operaciones básicas de un CRUD (alta, baja, modificación y listado/consulta) sobre registros de animales, aplicando una estructura organizada por clases para separar responsabilidades y hacer el código más mantenible.

Objetivo del proyecto:

EE objetivo principal de este proyecto fue aprender haciendo, en el marco del trabajo final de Programación II.
Más allá de lograr que funcionara, busqué que este proyecto me sirviera como una base real para practicar conceptos de desarrollo, enfrentar errores, validar datos, organizar mejor la lógica y comprender cómo construir una aplicación completa desde cero.
Este trabajo representa un avance importante en mi proceso de aprendizaje y en mi crecimiento como desarrollador, no solo por el resultado final, sino también por todo lo que aprendí durante el proceso.

Funcionalidades principales:

Alta de animales
Modificación de registros
Eliminación de registros
Listado de animales
Validaciones de datos
Manejo de tipos y estados mediante enums
Uso de listas para la gestión de registros
Persistencia de datos

Controller → controladores de la interfaz en JavaFX
Utils → lógica auxiliar, servicios, persistencia, helpers
Constantes → enums, textos y valores reutilizables
Exceptions → manejo de excepciones personalizadas
